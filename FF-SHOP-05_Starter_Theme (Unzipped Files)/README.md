# FF Shopify Paid Test Starter Theme

This is a sanitized hiring-exercise theme. It contains no production code, customer data, credentials, or approved production policy.

## Candidate task

1. Create your assigned branch.
2. Complete `sections/ff-shipping-regions-test.liquid`.
3. Complete `assets/ff-shipping-regions-test.css`.
4. Use the existing `templates/page.ff-paid-test.json` fixture data.
5. Run your normal theme validation and responsive QA.
6. Submit a pull request using the supplied PR template.

## Expected file scope

Most submissions should change only the two task files above. If another file must change, explain the reason in the PR.

## Safety

- Never connect this exercise to a production theme.
- Never paste credentials into code, screenshots, Loom, or pull-request comments.
- Do not add third-party libraries or trackers.
- This copy is a hiring fixture and must not be published.
